# 剧本资产拆解（Script Asset Breakdown）

> 将短剧/影视分镜脚本 PDF 自动拆解为 **场景卡 + 角色卡**，合并细碎镜头为可制作的大场景，并生成 AI 提示词用于影视概念图、分镜气氛图和置景参考图。

---

## 这是什么？

一个用于 **CodeBuddy Code** 的 AI Skill。上传你的分镜脚本 PDF，它会自动：

- 🔍 提取并分析剧本全文
- 🎬 将数十个细碎镜号**合并为可连续拍摄的大场景**
- 👤 为每个角色生成**角色卡**（含 AI 影视概念图提示词）
- 🎥 为每个场景生成**场景卡**（含 AI 分镜气氛图 / 置景参考图提示词）
- 📊 输出**统筹表**：场景调度、色调演进、核心道具清单

## 效果预览

输入：一份 12 页的 PDF 分镜脚本（13 个镜号）

输出：一份完整的 Markdown 文档，包含：

| 产出 | 数量 | 说明 |
|------|:--:|------|
| 角色卡 | 4 张 | 每张含身份/外貌/性格/人物弧光/AI概念图提示词 |
| 场景卡 | 10 张 | 合并后的可制作大场景，每张含分镜气氛图/置景参考图提示词 |
| 统筹表 | 3 张 | 场景调度总览 / 色调演进建议 / 核心道具清单 |

👉 [查看完整示例](references/example_breakdown.md)（基于《山Q说·缉毒警察篇》实际拆解）

## 安装方式

### 方式一：通过 CodeBuddy 技能市场安装

1. 下载 [script-breakdown.skill](https://github.com/yasin173707/script-asset-breakdown/releases/latest/download/script-breakdown.skill)
2. 在 CodeBuddy Code 中导入 `.skill` 文件

### 方式二：手动安装

```bash
# 克隆仓库到 CodeBuddy skills 目录
git clone https://github.com/yasin173707/script-asset-breakdown.git \
  ~/.codebuddy/skills/script-breakdown
```

### 方式三：本地开发安装

将整个 `script-breakdown` 目录复制到你的 CodeBuddy skills 目录下即可。

## 使用方法

1. 准备好你的分镜脚本 PDF 文件
2. 在 CodeBuddy Code 中上传 PDF 或拖入对话框
3. 输入以下任意指令：
   - `拆解一下这个剧本`
   - `帮我做剧本资产拆解`
   - `生成场景卡和角色卡`
4. AI 会自动执行拆解，输出到 `/workspace/《剧名》场景卡与角色卡.md`

## 拆解逻辑

### 镜头合并原则

> **同空间 + 同时间段 → 合并为一个可制作大场景**

例如：原脚本中"猫眼观察"（2个镜号）+ "隔墙偷听"（3个镜号）+ "敲门对峙"（5个镜号），因为它们都在同一走廊/门口空间、时间连续，合并为**一个大场景**，方便剧组在一个地点集中拍摄。

### 角色卡内容

- 身份、年龄、外貌特征
- 性格关键词、人物弧光
- 核心特质一句话概括
- 关联道具
- **AI 影视角色概念图提示词**（可直接用于 Midjourney / Stable Diffusion / DALL·E）

### 场景卡内容

- 原镜号映射、时长、空间、时间
- 核心情节节点
- 情绪基调解读
- 美术置景要点
- **AI 分镜气氛图 / 置景参考图提示词**

### AI 提示词规范

所有提示词默认用于以下三种用途：

| 类型 | 用途 | 标注 |
|------|------|------|
| 影视角色概念图 | 角色定妆参考 | `角色定妆参考 / 影视概念图` |
| 分镜气氛图 | 画面构图与氛围参考 | `分镜气氛图` |
| 置景参考图 | 场景搭建与美术参考 | `置景参考图` |

每条提示词均包含：构图角度、光线色调、镜头焦段、画幅比例、技术参数（`电影级写实风格，4K画质，35mm镜头`）。

## 适用场景

- 📺 短剧/网剧前期筹备
- 🎬 影视分镜脚本向美术组的交接
- 🎨 AI 影视概念图批量生成的前置准备
- 📋 制片统筹：场景调度规划、道具采购清单
